• Armazenagem De Paletes (Vertical)

  • Após o empilhador logar no sistema pelo coletor, na tela inicial clique em BUSCAR TAREFA.

  • Agora bipe na etiqueta de recebimento ou digite o código do palete no campo correspondente.

  • Na tela, verifique o endereço onde será armazenado o palete. Leve ao endereço indicado pelo sistema.

  • Agora é só bipar com o coletor o código da etiqueta do endereço ou digitar o código, onde o palete será armazenado e automaticamente finaliza o processo e retorna para a tela inicial para bipar o próximo palete.
