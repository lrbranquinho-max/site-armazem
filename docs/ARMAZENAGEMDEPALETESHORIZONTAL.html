• Armazenagem de paletes (Horizontal)

  • Após logar no sistema pelo coletor, na tela inicial clique em BUSCAR TAREFA

  • Agora bipe na etiqueta de recebimento ou digite o código do palete no campo correspondente.

  • Na tela seguinte, verifique o endereço onde será armazenado o palete. Leve ao endereço indicado pelo sistema.

  • Agora é só bipar com o coletor o código da etiqueta do endereço ou digitar o código, irá ser mostrado onde o palete foi armazenado e automaticamente finaliza o processo e retorna para a tela inicial para bipar o próximo palete.

Você pode levar dois paletes na transpaleteira de garfo duplo normalmente mas só poderá fazer uma atividade no coletor por vez. Após terminar uma movimentação, faça a outra na sequência. Só é permitido iniciar outra atividade depois que terminar a que já foi iniciada.
